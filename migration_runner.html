Database Migration Runner

This tool will run necessary database migrations for the autoshop application.

Add Permissions System

Creates permissions and role_permissions tables for role-based access control. Required for menu filtering by user roles.

Add opened_in_fina column to invoices table

Adds a column to track whether invoices have been opened in FINA system.

Create item_price_usage table

Creates a table to track most-used prices per item and vehicle for smarter price suggestions.
